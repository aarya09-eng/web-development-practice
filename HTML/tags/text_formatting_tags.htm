HTML (HyperText Markup Language) is the standard markup language used to create web pages. It defines the structure and content of a web page using a system of elements represented by tags. HTML is not a programming language; it is a markup language that tells browsers how to display content. Tim Berners-Lee invented HTML in 1991. The current standard is HTML5, released by the W3C and WHATWG.
